# AI 自学研究工作流（模板版）

> 适用对象：希望用 AI 建立系统化学习/研究流程的个人（考证、职业转型、兴趣研究等）。
> 关键理念：**结果导向 + 模块化流程 + 可沉淀资产**。

---

## 一、设定结果（Result First）
- **目标描述**：一句话写明本次学习要产出的成果（报告/课程/项目/考试分数）。
- **时间盒**：`起止日期 + 每日投入时长`（建议拆为「学习」「实践」「复盘」三段）。
- **评估标准**：列出 3 项可量化指标，例如：
  - 完成 12 个知识点闪卡，并通过 AI 自测正确率 ≥ 85%；
  - 每周输出 1 篇 800 字学习笔记，发布到小红书/公众号；
  - 模拟考试成绩 ≥ 120 分。

### Prompt：明确结果
```
你是一位学习项目规划师。请根据以下信息，输出「学习成果」「时间盒」「评估标准」三部分。
- 学习主题：{{主题}}
- 学习背景：{{已有基础 / 目标}} 
- 可投入时间：{{每天/每周时长}}
- 想要的最终产出：{{成果形式}}
```

---

## 二、资料获取（Gather）
1. **核心教材**：列出 3 类渠道（官方教材 / 权威课程 / 专家访谈）。
2. **高频问题**：让 AI 根据目标受众生成 10 个常见问题，用于指导检索。
3. **内容分类**：建议使用「概念、案例、公式、工具、误区」等标签。

### Prompt：构建资料清单
```
你现在是研究助理。围绕「{{学习主题}}」，请输出：
1. 核心教材（官方/权威/专家）各 2 个资源链接；
2. 10 个学习过程中最常见的问题；
3. 推荐的知识分类标签，用于整理笔记。
要求表格输出，并标注每条资源的推荐理由。
```

---

## 三、拆解学习（Learn）
- **拆模块**：按知识结构拆成 3~6 个单元，每单元包含「学习任务」「输入素材」「实践任务」。
- **AI 讲解**：将难点交给 AI，用「类比/图示/步骤」形式解释。
- **互动提问**：使用 Socratic Prompt 让 AI 反问自己，检验理解程度。

### Prompt：拆解+辅导
```
你是耐心的教学设计师。请针对「{{单元主题}}」产出：
1. 学习目标（3 条，动作动词开头）；
2. 关键概念解释（类比 + 示例 + 常见误解）；
3. 自测问答（5 条，混合选择/开放题），并附参考答案。
若内容涉及公式/图示，请输出文字描述或 ASCII 图表。
```

---

## 四、实践与复盘（Do & Reflect）
- **每日打卡模板**：
  - 今日完成（学习任务 + 产出成果）；
  - AI 提示或改进建议；
  - 次日安排（细化到任务）。
- **每周复盘**：
  - 本周成果（链接/截图）；
  - AI 教练提问（3 个 why 深挖）；
  - 阻碍 & 调整策略。

### Prompt：生成每日自律报告
```
你现在是我的学习教练。根据以下记录，输出「今日成果」「改进建议」「明日行动」三部分，每部分最多 5 条。
- 学习主题：{{主题}}
- 今日完成：{{任务清单/链接}}
- 遇到的问题：{{问题}}
- 感受：{{主观感受}}
```

### Prompt：周复盘
```
请以咨询顾问的视角，为「{{本周主题}}」写一份复盘报告，包含：
1. 佐证成果（引用具体数据/链接）；
2. 难点洞察与原因分析；
3. 下周策略（学习/实践/输出）；
4. 可沉淀的模板或 SOP 建议。
```

## 五、知识库沉淀（Document）
- **结构推荐**：
  - `00_Index`（概览 + 行动清单）
  - `10_Notes`（卡片式笔记，统一 YAML 头）
  - `20_Projects`（实践项目记录）
  - `30_Templates`（可重复使用的 Prompt/流程）
  - `40_Library`（资料归档）
- **卡片格式示例**：
```
---
Title: {{知识点标题}}
Category: {{概念/案例/公式}}
Source: {{资源链接}}
Date: {{YYYY-MM-DD}}
Tags: [AI, 自学, {{自定义}}]
---
# 概念
- 
# 示例
- 
# 应用场景
- 
# 常见误区
- 
```

### Prompt：自动生成卡片
```
请把以下文本整理成「概念/示例/应用/误区」结构的卡片，使用 Markdown 输出，并补充 2 个实际应用场景。
文本：{{学习记录或资源内容}}
```

## 六、可发布的产出（Publish）
- **快速拆小红书**：
  - 痛点开场 + 成果图（10 秒内说明结果）；
  - 步骤清单（3~5 点，每点配图/动图）；
  - 模板/Prompt 领取方式（评论关键词）。
- **公众号长文结构**：问题背景 → 方法论 → 流程 SOP → 案例 → 模板链接 → 行动建议。
- **变现思路**：
  - 发布「学习计划模板包」「打卡 Notion 模板」「知识库结构」；
  - 设立「30 天共学营」或 1v1 学习教练服务；
  - 整理课程/播客精选清单，作为引流赠品。
